# EPI

C++ implementation of problems from *Elements of Programming Interviews*.

## Setup

This project uses Clang++ on Windows with Visual Studio Code.

### Prerequisites

- Clang++ (LLVM-MinGW)
- Visual Studio Code with C/C++ extensions

### Building

Compile a source file:
```bash
clang++ filename.cpp -o filename
```

Run the compiled program:
```bash
./filename
```

Or use the Code Runner extension in VS Code.
